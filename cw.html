说你好

Class Widgets 是一款能够显示实时教学进度的桌面组件应用。其提供了美观的桌面组件和高度的自定义选项，助力于思考和探索未来中科技与教学的结合。

插件 & 主题
极致自定义

在 Class Widgets 中，提供了基于 Python 的插件系统和“插件广场”；你可以在“插件广场”中安装您需要的插件。以及彰显个性的主题系统，让你搭建出独一无二的 Class Widgets。

近乎完美的
兼容性

增强的通知系统

Class Widgets 提供了强大的通知系统，能够在上下课等时间点推出提醒；此外，插件也可调用通知接口，使其更加实用。

高度的自定义设置

Class Widgets 致力于提供高度的自定义选项，使每位用户能构建出独一无二的桌面课表。

浮窗

全新隐藏方式——浮窗，能够让 Class Widgets 的重要信息浓缩为一块浮动的窗口，更轻巧便捷。尽量减小其对教学工作的干扰。

Fluent Design (流畅设计)

Class Widgets 采用了由微软推出的 Fluent Design 设计语言，使界面更加简洁、舒适和流畅，且使用了 QFluentWidgets UI库。
